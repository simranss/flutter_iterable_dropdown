# Contributing to iterable_dropdown

First off, thank you for considering contributing! We value your time and effort.

## How Can I Contribute?

### Reporting Bugs
This project uses GitHub Issue Templates for bugs. Please click the "Issues" tab and select the **"Bug Report"** template. Fill it out as completely as possible to help us resolve the issue quickly.

### Suggesting Enhancements
If you have an idea for a new feature, please click the "Issues" tab and select the **"Feature Request"** template.

### Pull Requests (Contributing Code)
We actively welcome your pull requests.

1.  **Fork** the repository.
2.  Create your feature branch (`git checkout -b feature/amazing-feature`).
3.  Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4.  Push to the branch (`git push origin feature/amazing-feature`).
5.  Open a **Pull Request**.

Please make sure to update tests and documentation as appropriate.
